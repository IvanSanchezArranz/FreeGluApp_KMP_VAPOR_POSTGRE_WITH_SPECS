# Requirements: Real Vector Icons for Navigation

## Business Requirements
- Enhance the visual polish of the FreeGluApp user interface.
- Replace unstructured emojis with standard UI icons to ensure cross-platform consistency and visual accessibility.

## Acceptance Criteria
- [x] The app's bottom bar catalog uses the Material "List" or "Home" vector icon.
- [x] The app's bottom bar favorites uses the Material "Favorite" vector icon.
- [x] The FoodDetailScreen back button uses the Material "ArrowBack" vector icon.
- [x] The Favorite action toggle correctly visually distinguishes states with real icons.
